# Interactive Holographic Portfolio & Virtual CV

A modern, interactive portfolio and Virtual CV web application built with **React**, **Vite**, and **Lucide React**. Designed with a terminal command-line interface (CLI) and an interactive holographic academic/experience projector to showcase software engineering, business systems architecture, and data analytics projects.

##Key Features

* **Interactive Terminal CLI:** Execute live commands directly inside a simulated developer console to query experience, tech stack, and background.
* **Holographic Academic Matrix:** View degree credentials, course distinctions, and download official PDF transcripts dynamically.
* **Interactive Story Book:** Flip through multi-chapter background cards highlighting technical growth, academic milestones, and entrepreneurship.
* **Responsive & Accessible Design:** Styled with dynamic theme accents, custom CSS cards, and Lucide React iconography for high readability.

---

##Tech Stack

* **Frontend Framework:** React (Vite)
* **Styling:** Custom CSS3 (CSS Variables, Flexbox, CSS Grid)
* **Icons:** Lucide React
* **Hosting & Deployment:** Vercel / GitHub Pages

## Quick Start (Local Setup)

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v16 or higher) installed on your system.

### 1. Clone the Repository
git clone [https://github.com/LeeSeatile/my-portfolio.git](https://github.com/LeeSeatile/Portfolio-Website.git)
cd my-portfolio
npm install
npm run dev
